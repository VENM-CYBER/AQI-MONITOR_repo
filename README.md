🌍 Air Quality Monitoring System (AQI Monitor)

A smart and interactive Air Quality Index (AQI) Monitoring Web Application that provides real-time and predicted air quality data for different cities. The system visualizes pollution levels using charts and gives meaningful insights to users.

🚀 Features

🌆 City-wise AQI Data
Search and view AQI for multiple cities

📊 Interactive Charts
Visual representation of pollution trends (PM2.5, PM10, etc.)

🤖 AI-Based AQI Prediction
Predicts future AQI using machine learning models

🌈 AQI Status Indicator
Displays air quality levels like:

Good 🟢

Moderate 🟡

Unhealthy 🔴

🔍 Search Functionality
Quickly find air quality data for any city

🌙 Dark/Light Mode UI (if you added it)
